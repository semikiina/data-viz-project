# Data Visualization Project

This repository contains a data visualization project for university coursework. The goal is to analyze and present data insights using modern visualization techniques.

## Features

- Interactive charts and graphs
- Data exploration tools
- Responsive design

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/data-viz-project.git
    ```
2. Install dependencies:
    ```bash
    cd data-viz-project
    npm install
    ```

## Usage

Start the development server:
```bash
npm start
```
Open your browser at `http://localhost:3000` to view the visualization.

## Technologies

- JavaScript / TypeScript
- D3.js / Chart.js
- React
